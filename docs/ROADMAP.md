# Career Copilot Roadmap
# 产品长期规划（动态，但低频更新）
This document defines the long-term product roadmap.

Only update this document when the product direction changes.

Do not use this document to track daily development progress.

---

# Product Vision

Help every job seeker become Offer Ready through AI.

Career Copilot is not a resume generator.

Career Copilot is an AI-powered career assistant that helps users prepare for their next opportunity.
